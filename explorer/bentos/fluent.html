What's the difference between Acrylic and Mica?
Acrylic is a translucent, blurred material used for transient UI chrome — flyouts, menus, tooltips. Mica is an opaque, subtly wallpaper-tinted base used for window backgrounds. Acrylic floats; Mica grounds.
