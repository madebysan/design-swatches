FAQ
What is island architecture?
Islands ship interactive JS only for specific components, keeping the rest as static HTML for maximum performance.
Is Astro free to use?
Yes. Astro is fully open source under the MIT license with a thriving community of contributors.
Which UI frameworks work?
React, Vue, Svelte, SolidJS, Lit, Preact, and more via official integrations.
